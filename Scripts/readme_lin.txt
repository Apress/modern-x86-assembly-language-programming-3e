The default directory for the source code examples is ~/ModX86Asm3e/Code.
Adjust the directions that follow if you used a different directory.
See Appendix A for more information.

To run the shell scripts, you must set the file permissions as follows:

1. Open a terminal window in your home directory.

2. Type the following commands:
   cd ModX86Asm3e/Code/Scripts/lin
   chmod 0755 *.sh

-----------------------------------------------------------------------

Use ./make_chXX.sh to build chapter XX
Use ./make_all.sh to build all chapters

Use ./run_chXX.sh to execute chapter XX
Use ./run_all.sh to execute all chapters

Note: each run_chXX.sh script redirects its output to @run_chXX.txt
